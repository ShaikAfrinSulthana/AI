# Prerequisites

## System Requirements
- Python 3.8 or higher
- 4GB RAM minimum (8GB recommended)
- 2GB Free disk space
- Internet connection for API calls

## Required Accounts
- [Eneq](https://eneq.com/) account for LUM API access
- [Supabase](https://supabase.com/) account for database (if using cloud dataset)
